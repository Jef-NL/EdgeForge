# EdgeForge
IOT Framework for edge devices (Nodes) written in C++. A framework to simplify the setup of any IOT related project.

## Project structure
```bash
├─ include                      # Public headers
├─ lib                          # Library structure
│   ├── include                 # Internal Headers
│   │   ├── **/*.h
│   ├── src                     # Internal Source
│   │   ├── **/*.cpp
├── LICENSE                     # Apache License
├── README.md                   # This file
└── .gitignore
```
